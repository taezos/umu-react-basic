# umu-react-basic

## How to use
```
umu-react-basic: Generate react-basic-hooks Project

Usage: umu-react-basic COMMAND
  Use umu-react-basic to generate a scaffold for a react-basic-hooks project

Available options:
  -h,--help                Show this help text
  -v,--version             Show version

Available commands:
  init                     Initialize scaffold
```
